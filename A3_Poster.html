Introduction
This project predicts student performance using machine learning models by analyzing multiple academic, social, and personal lifestyle factors.

It is designed directly to evaluate final grades, providing actionable insights for early educational interventions and targeted student success strategies.
Dataset Information
  • Source: UCI Student Performance Dataset
  • Target Variable: Final Grade (G3)
  • Data Availability: 395 student records
  • Features (30+ combined): Encompasses daily demographics, study time, absences, family background, and previous academic grades (G1, G2).

Models Used
Two robust regression models were implemented and extensively compared to predict the continuous target score efficiently:
  • Linear Regression: establishes baseline linear relationships between student capabilities and final outcomes.
  • Decision Tree Regressor: effectively captures complex interactions and non-linear patterns.
